How-To Production Guide & Cheatsheet

1. The Tracking Alignment Rule

The engine matches inputs row-by-row. If you put 5 words in the Lyrics column, you must put 5 corresponding value elements inside the Beats and Pitch columns. Separate entries using single spaces or standard line breaks.

2. Rhythmic Values (Beats Column)

Use standard number values to build time signatures:
• 1 = Standard Quarter Note (Baseline pacing)
• 0.5 = Swift Eighth Note (Fast runs / Rapid delivery)
• 4 = Whole Note (Long, atmospheric vocal holds)
• 1.5 or 0.25 = Advanced floating syncopations

3. Pitch Commands (Pitch Column)

Bypass default configurations item-by-item using explicit keys:
• L / M / H = Low, Medium, and High default fields.
• do re mi fa so la di = Precise chromatic solfège notation nodes.

4. Studio Special Commands


• Vocal Breaks: Type an underscore _ in the Lyrics column to command a vocal rest. The band will play a background fill while the singer stays silent.
• Micro-Hesitations: Add a comma , or period . to the end of a word to add a subtle human pause after that syllable.
